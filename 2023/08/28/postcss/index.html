使用 postcss 插件

PostCSS 是一个用于转换 CSS 的工具，可以通过插件来实现将 px 转换为 rem。常用的插件包括 postcss-pxtorem 和 postcss-plugin-px2rem。配置插件后，它会自动将 CSS 文件中的 px 单位转换为 rem。

如果想实现整个项目自动将 px 转为 rem，主要有两个步骤

1.根元素挂载 font-size 样式
